# Validation Rules for Generated PDF Artifacts

## Structural Validation Rules
1. The main heading must appear in the generated PDF.
2. Expected section headings must be preserved.
3. Tables defined in the Markdown source must remain present in the PDF output.
4. Lists defined in the Markdown source must remain present in the PDF output.
5. Code block content must remain visible in the generated PDF.

## Content Validation Rules
6. Important keywords from the source document must be preserved.
7. Text content must not be truncated unexpectedly.
8. Expected section titles must remain unchanged.
9. Important values and labels must remain readable after conversion.
10. The generated PDF must not be empty.
